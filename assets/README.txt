Assets Folder
=============

Place your profile photo here with the filename: alkis-photo.jpg

Image Requirements:
- Filename: alkis-photo.jpg
- Format: JPG or WebP (JPG is simpler)
- Size: Optimize to less than 200KB for fast loading
- Dimensions: Recommended 400x400px (will be displayed as circular)
- Quality: High quality but compressed

Optional:
- You can also add a WebP version for better performance: alkis-photo.webp
- The HTML will automatically fall back to JPG if WebP is not supported

Image Optimization Tools:
- macOS: Preview (Export with reduced quality)
- Online: TinyJPG.com or Squoosh.app
- Command line: ImageOptim or imagemagick
